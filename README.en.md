# eext-delete-all-components

[中文](./README.md)

## Project Overview

eext-delete-all-components is a helper tool for batch deleting symbols, footprints, and components. It is suitable for scenarios where you need to remove all specified content under a certain ownership at once, such as components, footprints, etc.

## Features

- Specify a team or personal library to delete components, symbols, or footprints within it
- Support multi-select or delete specified content

## Safety Tips

- It is strongly recommended to back up before use. Go to the bottom component library - Batch Save As - Save As Local

## Installation (Example)

1. Download the extension:

   Go to the JLCPCB Extension Marketplace to download and install: [Installation Tutorial](https://prodocs.lceda.cn/cn/api/user-guide/using-extension.html)

## Usage Example

In the V2.2 editor, it is located in the top menu. In the V3 editor, it is in the Advanced menu. Click the "Delete All Components" menu to open a dialog, then select the desired ownership and corresponding library type to delete.
